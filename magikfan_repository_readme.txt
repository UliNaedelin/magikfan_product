Das öffentliche MagikFan Repository mit frei verfügbaren Tools und den Magik Beispielmodulen liegt in GitHub unter:

https://github.com/UliNaedelin/magikfan_product 

Der Branch exercise_solution enthält die Lösungen der Aufgaben-Module.


Das MagikFan Repository enthält außerdem folgende Hilfsmodule:
magikfan_abbreviation   	- nützliche Shortcuts, teils in Magik Schulung vorgestellt
magikfan_create_load_list   	- erzeugt und aktualisiert die load_list.txt automatisch
magikfan_debug_method   	- Dokumentiert Aufruf einzelner Methoden. Enthält Argumente, die Objekt-Instanz, den Rückgabewert sowie die Laufzeit.
